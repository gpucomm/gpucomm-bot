gpucomm-bot

GPU-aware automation and CI engine for the gpucomm organization.

Features

  • GPU / CUDA validation in PRs
  • CI checks for compute pipelines
  • Benchmark triggers before release
  • Issue and PR automation

Behavior

PR Detection

When a PR is opened with "GPU", "CUDA", or "NVIDIA" in the title:

  • Posts a comment indicating GPU-related status
  • Applies the gpu-required label

Issue Detection

When an issue is opened with "GPU", "CUDA", or "NVIDIA" in the title:

  • Posts a triage comment
  • Applies the gpu-required label

Release Events

Logs release events for benchmark triggering (Track B).

Data Handling

This bot processes GitHub webhook events containing:

  • PR titles and metadata
  • Issue titles and metadata
  • Release event data

No personal data is stored beyond GitHub's standard webhook payload.

Rate Limits

Subject to GitHub API rate limits. Uses installation tokens for authentication.
